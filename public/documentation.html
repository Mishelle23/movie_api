Hello and Welcome to my web application providing source of information about all sorts of different movies, including cast, director or genres.

Business Logic URL HTTP Method Request data format Response data format
Get list of all movies /movies GET None A JSON object holding complete data about all movies.
Return data about a movie by title. /movies/[Title] GET None A JSON object holding data about a single movie, containing description, genre, director and actor. Example:
id: 1,
title: "Jungle"
Description: "text"
Return data of all users. /users GET None A JSON object holding information of all users.
Return data about a single user using name. /users/[Name] GET None A JSON object holding data about a single user. Example:
Name: "anna lively",
Password: "word123",
Email: "annalively23@gmail.com"
Adding new user. /users POST A JSON object holding data containing users details: username, Password, email adress, birthdy. Example:
Username: "usertest",
Password: "xxxx",
Email: "usertst@post.com"
 Confirmation message whether the registration has been completed or not.
Allowing a user to update username. /users/[username] PUT A JSON object holding data conatining the new username. Example:


firstName: "New",
lastName: "Username"
 Confirmation message that a new username was succesfully created or not.
Removing user. /users DELETE A JSON object with details containing username and email adress, password and birthday. Example:
Username: "anyuser",
Password: "zzz", Email: "anyuser@post.com"
 A verification message if the deletion has been succesfull or not.
Adding a movie to the list of favorites using ID. /users[Username]/movies/[MovieID]/ POST A JSON object containing FavoriteMovies in form of MovieID.
Allow users to be updated /users/[Username] PUT A message confirmation about successfull update.
