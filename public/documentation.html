Documentation page

Hello and Welcome to my web application providing source of information about all sorts of different movies, including cast, director or genres.

Request URL HTTP Method Request data format Response data format
Get list of all movies /movies GET None A JSON object holding complete data about all movies.
Return data about a movie by title. /movies/[Title] GET None A JSON object holding data about a single movie, containing description, genre, director and actor.
Return data about a genre by name. /movies/genres[name] GET None A JSON object holding information of movies in a particular genre.
Return data about a director. movies/directors/[Name] GET None A JSON object holding data about a director.
Allow new users to register. /users POST A JSON object holding data containing users details: first name, last name, age and mail adress. Example:


firstName: "John",
lastName: "Smith",
age: "30",
mail: "John.S@hotmail.com"
 Confirmation message whether the registration has been completed or not.
Allowing a user to update username. /users/[username] PUT A JSON object holding data conatining the new username. Example:


firstName: "New",
lastName: "Username"
 Confirmation message that a new username was succesfully created or not.
Allow new users to deregister. /users DELETE A JSON object with details containing first name, last name and mail adress. Example:


firstName: "John",
lastName: "Smith",
mail: "John.S@hotmail.com"
 A verification message if the deregistration has been succesfull or not.
Allowing user to add a movie to the list of favorites. /movies/users/[User]/favorites PUT A JSON object which holds the name of the film to be added. Example:


name: "movieName"
 A message wether the film has been added succesfully to the favorites or not.
Allow users to remove a movie from their list of favorites. /movies/users/[User]/favorites DELETE A JSON object holding the name of the film to be removed. A message wether the film has been aremoved succesfully from the favorites or not.
